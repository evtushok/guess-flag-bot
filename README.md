# Бот Telegram Угадай Флаг

Telegram-бот для увлекательной игры в угадывание флагов стран мира. Пользователи видят флаг и должны угадать, какой стране он принадлежит.

Flag Guesser Bot — это интерактивный Telegram-бот, который помогает изучить флаги всех стран мира в игровой форме. Бот показывает случайный флаг, а пользователь должен выбрать правильный ответ из нескольких вариантов.

**Отлично подходит для:**
- Изучения географии
- Убийства времени
- Соревнования с друзьями
- Образовательных целей

---

## Требования

- Python 3.8+
- Токен Telegram Bot API

## Использование

### 1. Клонируйте репозиторий
```bash
git clone https://github.com/evtushok/guess-flag-bot.git
cd guess-flag-bot
```

### 2. Создайте виртуальное окружение
```bash
python -m venv venv
source venv/bin/activate  # Linux/Mac
# или
venv\Scripts\activate     # Windows
```

### 3. Установите зависимости
```bash
pip install -r requirements.txt
```

### 4. Запуск в режиме polling (для разработки)
```bash
python bot.py
```

### Команды бота

- `/start` — Начать игру
- `/help` — Помощь и правила

### Правила игры

1. Бот показывает флаг страны
2. Вам нужно выбрать правильный ответ из 4 вариантов
3. За правильный ответ начисляются очки
4. Старайтесь набирать наибольшее количество очков

- [aiogram](https://docs.aiogram.dev/) — Фреймворк для Telegram Bot API
- [aiohttp](https://docs.aiohttp.org/) — Асинхронный HTTP клиент
